## 功能

- [x] @置顶
- [x] 忽略点击
- [x] 开机启动
- [x] 小工具
- [x] 拖动（按住 Alt）
- [x] 放大缩小（Ctrl/Cmd 和 +、-、0 控制）
- [x] 提醒休息（默认 25 分钟）
